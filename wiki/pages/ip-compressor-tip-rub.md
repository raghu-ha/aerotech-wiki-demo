# IP Compressor Tip Rub

## Summary
Minor tip rub occurring in Stage 3 of the Titan 900 IP Compressor after 800+ flight cycles in high-humidity operating environments. Results in a measurable EGT rise above baseline.

## Technical Details
- Affected Assembly: IP Compressor Stage 3 (AT-T900-IPC-R001)
- Trigger Condition: 800+ flight cycles in high-humidity environments
- Impact: EGT increase of 5–8°C above baseline
- Detection: Borescope inspection
- Service Bulletin: SB-2023-031 (full bulletin not yet ingested)

## Current Status
Active issue. Monitored via borescope inspection. Addressed by SB-2023-031 (referenced in XA-001 maintenance log; full bulletin pending ingestion).
